📚 CSS Quick Reference Guide

A handy reference for CSS properties, Flexbox, and Grid

Flexbox Properties

Container Properties

display: flex;
flex-direction:
row | row-reverse | column | column-reverse
justify-content:
flex-start | flex-end | center | space-between | space-around | space-evenly
align-items:
stretch | flex-start | flex-end | center | baseline
flex-wrap:
nowrap | wrap | wrap-reverse
gap: 10px;

Item Properties

flex: 1; /* grow, shrink, basis */
flex-grow: 1;
flex-shrink: 1;
align-self:
auto | flex-start | flex-end | center | baseline | stretch
Remember: justify-content = main axis, align-items = cross axis

Grid Properties

Container Properties

display: grid;
grid-template-columns:
100px 200px 1fr;
grid-template-columns:
repeat(3, 1fr);
grid-template-rows:
100px auto 100px;
gap: 20px;
row-gap: 10px;
column-gap: 20px;

Grid Template Areas

grid-template-areas:
  "header header header"
  "sidebar main main"
  "footer footer footer";

Item Properties

grid-column: 1 / 3; /* start / end */
grid-row: 1 / 3;
grid-area: header;
Tip: Use fr units for flexible columns/rows

Common Patterns

Center with Flexbox

.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

Center with Grid

.container {
  display: grid;
  place-items: center;
}

Responsive Grid

.container {
  display: grid;
  grid-template-columns:
    repeat(auto-fit, minmax(250px, 1fr));
}

Card Hover Effect

.card {
  transition: transform 0.3s;
}
.card:hover {
  transform: translateY(-5px);
}

Responsive Design

Media Queries

/* Mobile First */
@media (min-width: 768px) {
  .container {
    max-width: 1200px;
  }
}
/* Desktop First */
@media (max-width: 768px) {
  .grid {
    grid-template-columns: 1fr;
  }
}

CSS Units

Absolute Units

Relative Units

Tip: Use rem for font sizes, px for borders

💡 Pro Tips

  • Always use box-sizing: border-box; for predictable sizing
  • Use Flexbox for one-dimensional layouts (rows OR columns)
  • Use Grid for two-dimensional layouts (rows AND columns)
  • Mobile-first approach is often easier (start small, add styles for larger screens)
  • Use CSS variables for maintainable code: --primary-color: #667eea;
  • Test your layouts in different browsers and screen sizes
