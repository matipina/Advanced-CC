# CSS Learning Journey 🎨

A comprehensive, hands-on lesson for learning CSS from the basics to advanced layout techniques including Flexbox and Grid.

## 📚 What You'll Learn

This interactive lesson covers:

1. **CSS Basics** - Selectors, properties, and how CSS works
2. **The Box Model** - Understanding margins, padding, borders, and box-sizing
3. **Flexbox** - One-dimensional layouts for flexible containers
4. **CSS Grid** - Two-dimensional layouts for complex designs
5. **Practical Example** - Building a modern dashboard step-by-step

## 🚀 Getting Started

1. Open `index.html` in your web browser
2. Follow the lessons in order from top to bottom
3. Each section builds on the previous one
4. Experiment with the code by opening the files in your code editor

## 📖 How to Use This Lesson

## 📝 Cheat Sheet

### Flexbox Quick Reference
```css
.container {
    display: flex;
    flex-direction: row | column;
    justify-content: flex-start | center | space-between | space-around;
    align-items: flex-start | center | flex-end | stretch;
    gap: 10px;
}
```

### Grid Quick Reference
```css
.container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    grid-template-rows: auto;
    gap: 10px;
}

.item {
    grid-column: 1 / 3;  /* Span 2 columns */
}
```
