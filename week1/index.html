1. CSS Basics

CSS (Cascading Style Sheets) is used to style and layout web pages. Let's start with the fundamentals!

Selectors

Selectors target HTML elements to apply styles:

Element selector: p { color: blue; }

ID selector: #unique { color: green; }

Class selector: .highlight { background: yellow; }

CSS properties control how elements look. Let's experiment!

2. The Box Model

Every HTML element is a box with content, padding, border, and margin.

Understanding the Box Model

Each element consists of: content → padding → border → margin

✏️ Adjust the padding, border, and margin values to see how they affect the box!

Box Sizing

box-sizing: content-box (default)

Width = content only

box-sizing: border-box

Width = content + padding + border

3. Flexbox - Flexible Box Layout

Flexbox is perfect for one-dimensional layouts (rows or columns).

Understanding Display Flex

Important: display: flex creates a flex container that affects only its direct children.

✏️ Try it yourself! Edit the CSS below and watch the preview update in real-time.

Display: Flex vs Inline-Flex

display: flex makes the container block-level, while display: inline-flex makes it inline.

✏️ Change display: flex to display: inline-flex and see the difference!

Direct Children Only!

Flexbox only affects direct children of the flex container, not nested elements.

✏️ Notice: Only the direct children (.child) are flex items. The .grandchild is not affected!

4. CSS Grid - Two-Dimensional Layout

Grid is perfect for complex two-dimensional layouts with rows and columns.

Basic Grid Layout

Create a grid by setting display: grid and defining columns.

✏️ Try different column configurations! Use 1fr, 2fr, 100px, or repeat(4, 1fr)
CSS (Edit Me!)
Live Preview

Grid Spanning - Making Items Larger

Use grid-column and grid-row to make items span multiple cells.

✏️ Change the grid-column values to make items span different amounts! Try 1 / 4 to span 3 columns.
CSS (Edit Me!)
Live Preview

Grid Template Areas (Named Grid)

Header
Sidebar
Main Content
Footer
.container {
    display: grid;
    grid-template-areas:
        "header header"
        "sidebar main"
        "footer footer";
}
.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
